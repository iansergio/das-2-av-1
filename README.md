# Projeto Integrador - VendaMais Distribuidora Ltda.
## Descrição do projeto
Nosso objetivo é desenvolver uma Plataforma de Inteligência Operacional para a empresa *VendaMais Distribuidora Ltda.*, integrar os dados do ERP do cliente com serviços em nuvem e realizar a construção para visualização de um dashboard estratégico.

## Instruções de navegação

### docs/adr/
Contém o registro das decisões de arquitetura do projeto.

- **adr-001.md**: documenta a decisão relacionada à estratégia de ingestão de dados

- **adr-002.md**: documenta a decisão relacionada à estratégia de armazenamento


### docs/c4/
Contém os diagramas do modelo C4, utilizados para representar a arquitetura do sistema em diferentes níveis de abstração.

- **01-context.md**: descreve o sistema no nível de contexto, apresentando os principais atores e sistemas externos e suas interações

- **02-container.md**: detalha os principais containers da plataforma de visualização, incluindo tecnologias utilizadas, responsabilidades e comunicação entre os componentes
